Asset Management Control

March, 2017 - April

Project Description

  • This project is based on vehicle(assets) auctions. Broadly, in this digital world, they would like to document everything from seller raising a hand to sell that and buyer wants to buy that asset. It has onsite auctions and online auctions as well in different locations across the world. I, majorly, worked on Asset management control.
  • In this, asset life cycle plays a vital role. It will be started from creating opportunity for the seller and it goes different phases such as moving that asset to contract ready, pre selling phases which involves inspection, adding asset details, photos to the project and eventually it dissolves with selling to another customer.
  • Apart from asset management control which I worked on mostly, there are another multiple projects like sale admin(which is about selling assets with stipulated things which will be managed by admin), site admin..etc.
  • It uses different technologies ranging from spring boot to angular(12+) to SQL. UI design look good with angular material modules everywhere which is the standard one for the total project. Service layer designed with java and spring boot.
  • In different application in this project we have much of repeated code so, they used different npm libraries. We need to write most of the code in libraries form and publish them into real code so that they can use for another projects too.
