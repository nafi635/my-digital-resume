Multi Recharge Application

March, 2017 - April

Project Description

  • Its web application where we can recharge with different options showing on the screen
  • Currently, we can see only some recharge options on different prices, for example: 11, 21, 31..etc. But we are providing options to recharge almost on every amount. We provide recharge option as low as 5 rupees.
  • It is done using JSP, Java and boostrap majorly.
  • This entire web application is completed in 2 months as a part of major project during my college days.
