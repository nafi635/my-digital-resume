Seller Experience Portal

Project Description

  • It is also one of the projects of Ritchie bros, vehicle auction selling business, It the separate application for the sellers who sell vechicles at Ritchie Bros. Auctioneers
  • It includes dashboard for a sellers, actions for sellers, invoices report and sale adjustment tools who come to sell their manufacturing units, machineries, vehicles. And also it consists of some stats like how many assets are selling or how many assets they sold.
  • It uses Angular for UI. This project is deployed over AWS. Graph QL queries used to get the data from Open search schema of AWS.

Tech Stack

Java • Spring Boot
Angular • Typescript • Angular Flex • Angular Material • Figma UX
• Oracle • Graph QL •

Responsbilities

  • I got into completely new end to end project where I was the only one working for UI development.
  • Business gives UI representation or Sketch through Figma application frames. It was quite daunting task as they were demanding exact inch-to-inch design with even width and height of the elements should be same.
  • API implmentation - I had to implement multipl API's based on the requirement and calling them through the Graph QL.
  • Along with them, I need to connect to UI from open search(Elastic search) data base and need to write Graph QL queries for getting a data.
